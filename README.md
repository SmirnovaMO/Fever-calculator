# Калькулятор жаропонижающих для детей

Веб-версия Flutter приложения для расчета дозировки жаропонижающих препаратов для детей.

## Доступ к приложению

Приложение доступно по адресу: [https://yourusername.github.io/fever-calculator/](https://yourusername.github.io/fever-calculator/)

## Функционал

- Расчет дозировки жаропонижающих препаратов
- Ведение истории температуры
- Уведомления о приеме лекарств
- Графики температуры
- Экспорт данных в PDF

## Технологии

- Flutter Web
- Dart
- GitHub Pages
