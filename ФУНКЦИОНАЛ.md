# Калькулятор жаропонижающих - Описание функционала

## Общее описание
Flutter-приложение для расчета дозировок жаропонижающих препаратов для детей с учетом возраста и веса.

## Основные функции

### 1. Управление семьей
- **Добавление членов семьи** с указанием имени, возраста и веса
- **Проверка корректности веса** по возрастным нормам:
  - 1-11 месяцев: индивидуальные диапазоны для каждого месяца
  - 1-5 лет: диапазоны по годам (8-13кг до 14-24кг)
  - 6-17 лет: диапазоны по годам (16-27кг до 46-90кг)
- **Редактирование** данных членов семьи
- **Удаление** карточек
- **Сохранение данных** локально через SharedPreferences

### 2. Мониторинг температуры
- **Запись температуры** с автоматическими рекомендациями по возрастам:
  - **До года (0-11 мес)**: детальные диапазоны от 35°C до 41°C+
  - **1-5 лет**: адаптированные рекомендации с учетом подгузников
  - **6-17 лет**: рекомендации без упоминания подгузников
  - **Старше 17 лет**: базовые предупреждения
- **График температуры** с отметками приема лекарств
- **Проверка интервалов** - предупреждение при измерении раньше 1.5 часов после приема лекарства

### 3. Система лекарств
#### Ибупрофен (5 форм):
- **Суспензия 100мг/5мл** (3-40кг): минимальная/оптимальная дозировка в мл
- **Суспензия Форте 200мг/5мл** (3-40кг): минимальная/оптимальная дозировка в мл  
- **Свечи 60мг** (3-13кг): фиксированная дозировка в свечах
- **Свечи 125мг** (7-29кг): фиксированная дозировка в свечах
- **Таблетки 200мг** (10-40кг): рекомендуемая/максимальная дозировка

#### Парацетамол:
- Пока не реализован (заготовка есть)

### 4. Безопасность дозирования
- **Автоматический расчет** рекомендуемых доз по весу ребенка
- **Проверка превышения** максимальной разовой дозы
- **Контроль суточной дозировки** с предупреждениями
- **Интервалы между приемами** (6-8 часов)
- **Предупреждение о делении свечей** для нецелых доз
- **Конвертация в миллиграммы** для точного учета

### 5. Интерфейс
- **Выпадающие списки** для выбора форм препаратов
- **Поля ввода** с проверкой корректности
- **Цветовая индикация** (зеленый - норма, оранжевый - внимание, красный - опасно)
- **История наблюдений** с временными метками
- **Адаптивный дизайн** для веб-платформы

### 6. Образовательный контент
- **Статья 1**: "Важная информация при повышении температуры" (отредактирована пользователем)
- **Статья 2**: "Первая помощь, если у ребенка судороги"

## Технические особенности
- **Платформа**: Flutter (веб-версия)
- **Хранение данных**: SharedPreferences (локально)
- **График**: FL Chart библиотека
- **Интернационализация**: Intl пакет для форматирования дат
- **Архитектура**: StatefulWidget с разделением на экраны и модели

## Файловая структура
```
lib/
├── main.dart - точка входа
├── models/
│   ├── child.dart - модель ребенка с расчетами доз
│   └── article.dart - модель статьи
├── screens/
│   ├── main_screen.dart - главный экран с семьей и статьями
│   ├── child_screen.dart - экран ребенка с лекарствами и температурой
│   └── article_screen.dart - экран просмотра статьи
└── widgets/
    └── temperature_chart.dart - виджет графика температуры
```
